(정적 프로그램 언어인 html을 동적으로 제어하기 위한 언어)

- github  
버전관리 웹 서비스  
로컬에 설치된 sourcetree 프로그램을 통해 visual studio code에서 작성한 코드를 github 프로젝트에 push를 함으로써 저장한다.    
<순서>  
visual studio code에서 코드 작성 -> sourcetree에서 commit(로컬 저장소에 저장) -> push(github 사이트에 저장)
